简述S3

Amazon S3 是 Internet 数据的存储库。Amazon S3 提供了可靠、快速和廉价的数据存储基础设施。它的目的是通过支持您随时从 Amazon EC2 内部或从网络上的任何地方存储和检索任何数量的数据来简化整个网络计算。Amazon S3 以冗余方式跨多个设施在多个设备上存储数据元，允许多个不同的客户端或应用程序线程同时对这些数据元进行读或写操作。您可以使用存储在 Amazon S3 中的冗余数据快速、可靠地恢复实例或应用程序故障。有关更多信息，请参阅 Amazon Simple Storage Service (Amazon S3)。


使用S3

  1. 注册亚马逊账号，在控制中心开通S3，并创建存储桶，创建对象。具体创建流程，参阅Amazon S3。

  2. 到GitHub下载官方SDK。


从客户端SDK授权

  1. 配置Cognito管理端。

    • 进入亚马逊控制中心在服务中选择Cognito。
    • 选择创建新标识池Create new identity pool。
    • 填写Identity pool name，勾选Enable access to unauthenticated identities，点击创建进入角色创建页面，点击允许完成创建。

  2. 进入IAM，选择角色，点击创建池时创建的授权角色，进行策略编辑，修改如下：

  3. 通过方法initWithRegionType:identityPoolId:创建授权凭证。

    • regionType：S3服务区，同S3控制中心服务区一致。
    • identityPoolId：Cognito管理端的唯一标识池。

    ​

  4. 创建默认服务配置，完成授权。

    • region：S3服务区，同第一步中的服务区一致。

    • sessionKey：令牌
    • expiration：失效时间

  1. 构建凭证，配置服务完成后，进行文件上传。首先进行上传请求。
