Braintree集成指南

  • Braintree
  • 境外支付

为什么使用Braintree

  • 集成简单
  • 自定义UI
  • 一键支付
  • 多渠道支持等

使用Braintree的条件

  • 公司必须在Braintree服务地

  • 必须有所在服务地的公司银行账号

    费用

    美国、香港为例，其他地区详见其开发者网站

  • 美国：2.9% + $.30 per transaction

  • 香港：3.40% + 2.35 HKD per transaction

账号注册

  • 注册PayPal企业账号
  • 注册Braintree沙盒账号
  • 注册Braintree生产环境账号

相关说明

  • PayPal和Braintree关联后，PayPal账号可以更换，但是会丢失对应PayPal的客户数据，因为Braintree里的数据是从PayPal里导入的。
  • 用户使用Visa、万事达卡支付，2天之内到账，美国运通4天之内，PayPal支付通常在24h之内(保守)。

工作原理

  1. 客户端向自身服务器发送token请求来初始化客户端SDK。
  2. 服务器收到请求后，生成token返回给客户端SDK。
  3. 一旦客户端SDK被初始化，并且客户已提交支付信息，客户端SDK会向Braintree发送消息，Braintree会返回给客户端SDK一串随机数。
  4. 客户端SDK将随机数发给自身服务器。
  5. 自身服务器将收到的随机数通过服务器SKD去创建交易。

  4. 注册沙盒账号

iOS端配置

  1. Braintree SDK下载

  2. 适配 iOS 9

    在Info.plist中添加LSApplicationQueriesSchemes字典，并添加com.paypal.ppclient.touch.v1, com.paypal.ppclient.touch.v2,org-appextension-feature-password-management, 如果集成了venmo需要添加com.venmo.touch.v2。

  3. 向自身服务器请求token

  4. 向自身服务器发送交易随机数

  5. 自身服务器通过客户端传来的交易随机数创建交易闭环
